Introduction

This guide walks you through installing Arch Linux With a script that i made...


the youtube tutorial is below

https://youtu.be/uwNUb4c5P-k?si=VzJf4f7fpPC1JXuq

Steps

  1. Boot from the Arch ISO.
  2. Partition the disk with fdisk you can check your drive name with lsblk
  3. Then install git with pacman -Sy git
  4. Clone the shelly installer script with git clone https://github.com/n0b0dy-arch-btw/ArchLinux-Installer.git
  5. cd into the downloaded file type ls to list then cd into the file
  6. Configure the install.conf to your needs
  7. then run chmod +x arch_install.sh to make the file executable
  8. now you can run the installer with ./arch_install.sh
  9. now you have arch installed and you can reboot
  10. once you have loged into your system you will need to install git with pacman -Sy git
  11. then cd into the installed file with cd
  12. now you need to configure the postinstall.conf to your systems needs
  13. make the post_install.sh executable with chmod +x post_install.sh
  14. now you can run the install with post_install.sh
  15. now your arch system is all done and should be running perfect
  16. if you have a issue then comment on the youtube video the link is under the introduction
